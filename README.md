# Food-Order-App
> A **ReactJS** based Food Order App using various concepts learned so far including :
- useState() hook
- useReducer() hook
- useEffect() hook
- custom hooks
- Module css classes
- useContext() hook
- React Forms and validation
- FireBase DataBase
